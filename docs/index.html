fastiron

Fastiron is a port of the Quicksilver mini app in Rust. It mimics Monte-Carlo particle transport simulation codes to study their behavior on various hardware architectures.

running fastiron

Executing the binary with no arguments will print the help message for a detailed CLI breakdown. For further explanation, refer to the documentation.
